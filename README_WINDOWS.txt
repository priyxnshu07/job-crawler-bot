================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Python is not recognized"
Solution: 
- Install Python from https://www.python.org/
- During installation, check ‚úÖ "Add Python to PATH"
- Restart computer
- Try again

Problem: "Virtual environment not found"
Solution:
- Run SETUP.bat first
- Wait for it to finish
- Then run RUN.bat

Problem: Files open in Notepad
Solution:
- Right-click the .bat file
- Select "Run as administrator"
- OR use Command Prompt (see below)

Problem: "Windows protected your PC"
Solution:
- Click "More info"
- Click "Run anyway"
- This is normal - Windows is just being cautious

================================================================================
USING COMMAND PROMPT (Alternative Method)
================================================================================

If double-clicking doesn't work:

1. Press Win + R
2. Type: cmd
3. Press Enter
4. Type: cd "C:\path\to\your\project"
   (Replace with your actual project folder path)
5. Press Enter
6. For setup: type SETUP.bat and press Enter
7. For running: type RUN.bat and press Enter
